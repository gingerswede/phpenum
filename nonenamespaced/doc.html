How to create an enumeration.

PHP lack the ability to create enumerations. This can, if you are used to them, make programming a bit difficult. There are ways to emulate this but that requires the latest PHP release and that that library is installed on your server.

This extention makes it possible to almost emulate the behavior of an enumeration. What you do is that you create a php-class containing constants. The constant name is the named part of an enumeration and then you assign the number you want that name to hold and then extend it with the PurpleKiwi Enum class Exampel: 

<?php
require_once '/path/to/purplekiwienum.php';

class Enum
extends PurpleKiwiEnum{
	const January = 1;
	const February = 2;
	const March = 3;
	const April = 4;
	const May = 5;
	const June = 6;
	const July = 7;
	const August = 8;
	const September = 9;
	const October = 10;
	const November = 11;
	const December = 12;
}

Now you have access to all the functions listed in the menue to work with this class in an enumeration fashion. Exampel: 

$html = "<select name='months'>\n";
						
foreach (Enum::GetNameNumberPair() as $name => $number) {
	$html .= "\t<option value='$number'>$name</option>\n";
}

$html .= "</select>\n";

return $html;

Previous example would return: 

<select name='months'>
	<option value='1'>January</option>
	<option value='2'>February</option>
	<option value='3'>March</option>

	<option value='6'>June</option>
	<option value='7'>July</option>
	<option value='8'>August</option>
	<option value='9'>September</option>
	<option value='10'>October</option>
	<option value='11'>November</option>

public static function GetNames()

Return

Returns all constants as an array where the value is constant name and key is the corresponding value.

public static function GetName($number)

Param

int $number The number part of an enumeration.

Return

Return a specific value associated with that number.

public static function GetNumber($name, [$casesensitive = true])

Param

string $name The named part of the enumeration.

Param

bool $casesensitive This declares if the search should be case sensitive or not.
True - Search is case sensitive. (Default)
False - Search is not done case sensitive.

Return

Number of that named element.

public static function GetNameNumberPair()

Return

Array where the keys are the names and the value is the enumeration number.

public static function HasNumber($number)

Param

int $number Number to see if exists.

Return

bool

public static function HasName($name)

Param

string $name Named entity to search for. Case sensitive.

Return

bool

public static function HighestEnumeration()

Return

Number value of the highest numbered enumeration.

public static function LowestEnumeration()

Return

Lowest number of the enumeration.
